Selected Projects

A curated collection of projects that I'm proud of.

AC-PINN

AC-PINN is a physics-informed neural network framework that improves PDE solving under sparse and noisy data using residual-based curriculum learning and adaptive loss weighting. By dynamically controlling both sampling and optimization, it achieves more stable convergence and higher accuracy than standard PINNs.

Python PyTorch NumPy Matplotlib
📖 Details ⬡ GitHub

Automated UWB Bistatic Localization using CIR-Based Ellipsoid Modeling in Blender

Developed an automated Ultra-Wideband (UWB) localization pipeline by integrating Python-based add-ons within Blender. The system processes Channel Impulse Response (CIR) data to extract signal peaks, compute bistatic path lengths, and dynamically generate intersecting ellipsoids for real-time position estimation in a simulated vehicle environment.

Python Blender Blender Python API Computational Geometery UWB Signal Processing Concepts
📖 Details ⬡ NDA

LHC Jet Anomaly Detection

Unsupervised anomaly detector trained on QCD jets from LHC collision simulations to flag W boson, Z boson, and top quark jets as new physics candidates using reconstruction error.

Python PyTorch UMAP scikit-learn HLS4ML Jet Datase
